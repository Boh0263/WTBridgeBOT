# WhatsApp 및 Telegram 계정을 연결하는 방법

**참고: 크로스 플랫폼 멘션은 현재 개발 중이며 올바르게 작동하지 않을 수 있습니다. 연결 프로세스는 작동하지만 플랫폼 간 멘션 알림에는 알려진 문제가 있습니다.**

이 가이드는 WhatsApp 및 Telegram 계정을 연결하여 멘션(@태그)이 두 플랫폼 모두에서 작동하도록 하는 방법을 설명합니다. 일단 연결되면 한 플랫폼에서 누군가를 태그하면 다른 플랫폼에서 알림을 받습니다!

## 전제 조건

- BridgeBOT이 실행 중이고 WhatsApp 및 Telegram 그룹을 연결하고 있습니다.
- 두 그룹 모두에 계정이 있습니다.
- 짧은 이름이 설정되었습니다(아래 참조).

## 단계 1: Telegram에서 짧은 이름 설정

먼저 고유한 짧은 이름(1-9자의 영숫자 문자, 공백 없음)을 선택합니다.

1. Telegram 그룹으로 이동합니다.
2. 보내기: `/link <귀하의 전화번호> <짧은이름>`
   - 예: `/link 1234567890 john`
   - 전화번호: 국가 코드를 포함하고 + 또는 공백 없음.
3. 봇이 확인 또는 오류로 응답합니다.

귀하의 짧은 이름은 이제 Telegram 계정에 연결되었습니다.

## 단계 2: WhatsApp에서 연결

1. WhatsApp 그룹에서 보내기: `!iam <짧은이름>`
   - 단계 1의 동일한 짧은 이름을 사용합니다.
   - 예: `!iam john`

2. 봇이 귀하의 Telegram 계정에 확인을 요청하는 개인 메시지를 보냅니다.

3. Telegram(봇과의 개인 채팅)에서 응답: `yes`
   - 이는 30초 이내에 완료되어야 합니다.

4. 두 앱 모두에서 확인 메시지를 받습니다.

## 단계 3: 연결 테스트

**참고: 현재 제한으로 인해 크로스 플랫폼 멘션이 예상대로 작동하지 않을 수 있습니다.**

- WhatsApp에서 누군가를 태그한 메시지 보내기: `@그들의 짧은이름`
- Telegram에 나타나야 합니다(멘션 기능이 제한될 수 있음).
- Telegram에서 누군가를 태그한 메시지 보내기: `@그들의 사용자이름`
- WhatsApp에 나타나야 합니다(멘션 기능이 제한될 수 있음).

## 문제 해결

### "일치하는 Telegram 사용자를 찾을 수 없음"
- 귀하의 짧은 이름이 올바르고 고유한지 확인하십시오.
- Telegram 사용자가 `/link`로 짧은 이름을 설정했는지 확인하십시오.

### "확인 만료됨"
- 30초 창이 지났습니다. `!iam <짧은이름>`을 다시 시도하십시오.

### "전화번호가 이미 연결됨"
- 다른 사람이 그 번호를 사용했습니다. 필요한 경우 먼저 `/unlink`를 사용하십시오.

### 멘션이 작동하지 않음
- 크로스 플랫폼 멘션은 현재 실험적이며 버그가 있을 수 있습니다.
- 두 사용자가 모두 연결되었는지 확인하십시오(연결은 작동하지만 멘션은 작동하지 않을 수 있음).
- 오류에 대한 봇 로그를 확인하십시오.
- 매핑이 로드되지 않으면 봇을 다시 시작하십시오.

### 명령이 작동하지 않음
- 올바른 위치에 명령을 보내고 있는지 확인하십시오:
  - `/link` 및 `yes`: 봇과의 Telegram 개인 채팅에서.
  - `!iam`: WhatsApp 그룹에서.

## 연결 관리

- **상태 확인**: 직접 명령 없음, 멘션으로 테스트하십시오.
- **연결 해제**: Telegram 개인 채팅에서 `/unlink`를 보내고 확인을 위해 `yes`로 응답하십시오.
- **짧은 이름 변경**: 먼저 연결을 해제한 다음 새 짧은 이름으로 연결하십시오.

## 팁

- **짧은 이름 규칙**: 1-9자, 문자와 숫자만.
- **전화 형식**: 10-15자리 숫자, 기호 없음(예: 미국의 1234567890).
- **개인정보**: 크로스 플랫폼 멘션이 작동하려면 연결이 필요합니다.
- **그룹 관리자**: 봇이 메시지를 읽을 권한이 있는지 확인하십시오.

## 지원

문제가 발생하는 경우:
1. 모든 단계를 두 번 확인하십시오.
2. 봇이 온라인인지 확인하십시오: 메시지가 전달되는지 확인하십시오.
3. 그룹 관리자 또는 봇 로그에 문의하십시오.

이제 플랫폼 간에 친구를 원활하게 멘션할 수 있습니다! 🎉
